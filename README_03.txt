É UMA PERGUNTA DE NEGÓCIOS RESPONDIDA ATRAVÉS DE ALGUMA ANÁLISE DE DADOS ESPECÍFICAS 

MÉTODO ABCDE PARA ORGANIZAÇÃO NA CRIAÇÃO DE UM DASHBOARD
A- Assets (recursos: imagens, vídeos, gifs, paletas de cores de cores)
B- Bases (dados para gerar a dashboard principal ou outras medidas)
C- Cálculos (meio de campo que processa dados para transformar em informações – salvar cálculos, fórmulas, fórmulas dax, métodos)
D- Dashboard (painel visual propriamente dito)
E- Extras (estudos, procedimentos, requisições adicionais)

Criação de um dashboard no Excel partindo de uma base de dados de uma loja de jogos da Xbox, fornecendo informações relevantes.
1- Conhecer e entender a base de dados (base de dados de assinatura da Xbox Cloud);

2- Estabelecer a pergunta de negócio, que deve ser respondida através de alguma análise dos dados que possa ser convertida e apresentada como informação: 

a. PERGUNTA DE NEGÓCIO 1: Qual o faturamento TOTAL DE VENDAS de PLANOS ANUAIS (contendo todas as assinaturas agregadas)?
b. PERGUNTA DE NEGÓCIO 2: Qual o faturamento total de vendas de planos anuais separados por auto renovação e pelo que não é auto renovação.
c. PERGUNTA DE NEGÓCIO 3: Total de vendas de assinaturas do EA Pass.
d. PERGUNTA DE NEGÓCIO 4: Total de vendas de assinaturas do Minecraft Season Pass.
e. PERGUNTA DE NEGÓCIO 5: Informar a composição de cada período de assinatura em relação ao total.

3- Criar as consultas aos dados necessários para responder às perguntas de negócio

4- Ao criar o dashboard, é necessário dar sentido e contextualizar as informações que devem ser claras e intuitivas.

5- A organização do dashboard deve dar ênfase às informações

6- É uma boa prática informar a data de apuração dos dados.
