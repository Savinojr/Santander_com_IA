      O primeiro passo é definir o objetivo do projeto, que seria criar uma ferramenta para auxiliar na organização dos dados necessários para realizar a declaração de IRPF.
      Definimos quais informações iríamos priorizar, e assim sendo, foram criadas 3 planilhas: TITULAR, INFORMES E NOTAS.
      Criamos um menu para facilitar a navegação entre as diferentes seções do aplicativo, utilizando ferramentas de design, de bloqueio de alterações, criação de links internos e externos para os botões, replicando o layout para todas as planilhas.
      Após definir quais dados seriam obrigatoriamente preenchidos, realizamos algumas formatações de células personalizadas para adequar o preenchimento correto segundo o tipo do dado e também listas de validação de dados para que os dados fosses normalizados.
      Para finalizar, inseri um gráfico de composição de saldo bancário e protegi as áreas não editáveis da ferramenta.
